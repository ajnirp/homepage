Projects

These are some projects I've worked on. Click on a project name for more information on it, click again to hide.

Monopoly - the game

A computer game version of the business simulation board game Monopoly for multiple human players. Code written in C++ for my Intro to Programming course project.

Chinese Checkers

A computer game version of the board game Chinese Checkers. Supports human vs human (upto 6 human players), human vs AI or AI vs AI modes in which the user sits back and watches. You can also choose the precise AI algorithm to be used (Minimax, or Alpha-Beta pruning, or Randomized Play). Code written in Racket for my Abstractions and Paradigms in Programming course project.

Conway's Game of Life

Program that runs a fixed nunmber of iterations of the classical Conway's Game of Life with a randomized starting board and then exits. Code written in Python.

FMoT (File Manager on Terminal)

Terminal-based file browser utility for Linux systems inspired by programs like mocp and ncmcpp. Provides the functionality of a standard file browser, including browsing directories and opening, cutting, copying, pasting, renaming and deleting files. The program is controlled by the keyboard, with intuitive controls and a help system. Code written in C++ for my Data Structures and Algorithms Lab course project.

Rube Goldberg Machine Simulator

Realistic simulation of a Rube Goldberg Machine. A Rube Goldberg machine is a series of mechanical apparatuses that transfer energy among each other in some contrived way to finally execute a simple action. Code written in C++ with rendering and GUI in OpenGL. Used Box2D as the physics engine. Written for my Software Systems Lab course project.

Fast Fourier Transform on FPGA

Eight-point radix-2 DIF (Decimation in Frequency) Fast Fourier Transform implemented on Atlys Spartan 6 FPGA. User flips the DIP switches on the Spartan board to indicate input and presses a push button to input the next value. Output displayed on the LEDs above the DIP switches, separated by presses of the push button. Project code written in VHDL for my Logic Design Lab course project.

Named Entity Recognition

Given a text file containing the text of a novel, my program parses and identifies to a high degree of accuracy the various characters in the novel; their names, genders and status (protagonist, antagonist etc.). This problem is a subproblem for Named Entity Recognition, a fundamental natural language processing problem. Code written in Java.

Image Morphing

Implemented the Beier-Neely image algorithm for morphing one image into another with a GUI interface. Can specify the number of frames, the morph lines to be used, and the two images which we are changing between. Code written in Java.

RTalk - Anonymous Web-based Chat

Web-based chat app made as part of my internship at Chronus Corporation. Users start a private chat room (no sign-in/authentication of any kind) and invite their friends over to chat by sharing the randomized room URL. The chat room and all the associated chat logs are destroyed once everyone has left the room. Features include: ability to export chat history as as a text file, emoticon support, clickable links in-chat. Code written in Ruby on Rails.
